# 发布说明 V1.6.1765

## 版本信息
- 版本号：V1.6.1765
- 打包时间：2025-12-12

## 更新内容
1.  **新增功能：采购任务信息修改**
    - 支持对已生成的采购任务进行关键信息修改。
    - **常规修改**：支持修改“采购任务名称”和“需求单位”，修改后立即生效，不影响单号。
    - **高级修改**：支持修改“标的类别”和“计划月份”。
        - 系统会自动重新生成符合新规则的**采购任务单号**（例如：从 `2601MP-...` 变更为 `2602MPJ-...`）。
        - 自动占用新的流水号，保证单号唯一性。
        - 自动更新该任务下所有采购明细的序号前缀。
        - 自动更新相关的下达记录关联。
    - **操作入口**：在主界面表格中选中任务 -> 右键点击 -> 选择“修改任务信息”。

## 部署说明
1.  **全新部署**：
    - 直接运行 `PPOMS_V1.6.1765.exe`。
    - 目录下的 `purchase.db` 为初始空数据库（保留了基础配置：部门、状态等），可直接使用。
2.  **升级部署**：
    - 将 `PPOMS_V1.6.1765.exe` 复制到原程序目录。
    - **不要覆盖**原有的 `purchase.db`，否则会丢失历史数据。
    - 若需要使用新功能，直接运行新版 exe 即可。

## 数据库状态
- 本发布包中的 `purchase.db` 已执行“清空测试数据”操作。
- 保留了基础配置表（单位、采购员、状态、计划月份、打印配置等）。
- 移除了所有订单、明细及下达记录。
